# ChatConnect – Real-Time Chat Application

## 🚀 Overview
ChatConnect is a real-time messaging application built using the MERN stack and Socket.io, enabling users to communicate instantly.

---

## 🔧 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Real-time: Socket.io

---

## ✨ Features
- Real-time messaging
- JWT-based authentication
- Multiple user support (50+ users)
- Low latency communication (<200ms)
